# DunRay

**DunRay** (also referred to as "ReGen Insight") is an AI-powered platform designed to accelerate environmental regeneration across Kenyaâ€™s 47 counties. Its main focus is on land restoration, water conservation, and transparent carbon credit verification.

## Key Features

- **Comprehensive Coverage:**  
  Real-time environmental data monitoring for all 47 Kenyan counties.
- **AI-Powered Analytics:**  
  Uses advanced AI/ML (TensorFlow, OpenAI) to recommend optimal tree species and estimate carbon sequestration potential.
- **GIS and Satellite Imagery:**  
  Leverages satellite data for NDVI (greenness), land degradation, water stress, and more.
- **Blockchain Verification:**  
  Integrates with the Polygon network to provide an immutable ledger for carbon credit verification.
- **Community Engagement:**  
  Supports image uploads and crowdsourced data for ground-truthing and transparency.
- **Water Stress Analytics:**  
  Provides monthly, county-specific water stress data and analytics.

## Technology Stack

- **Frontend:** React, Tailwind CSS  
- **AI/ML:** TensorFlow, OpenAI  
- **GIS:** Satellite imagery analysis  
- **Blockchain:** Polygon Network

## Main UI Tabs

- **Dashboard:** Overview of platform data and user authentication.
- **GPS Map:** Interactive map for geospatial data visualization.
- **Analytics:** Charts and graphs for environmental metrics.
- **Community Images:** Upload and view images contributed by users.
- **Ledger:** Blockchain-based carbon credit verification.

## Data Model

- Each county includes:
  - Name, coordinates, NDVI, land degradation level, water stress, carbon sequestration potential.
  - Recommended tree species, soil type, rainfall, and area.
- Monthly water stress analytics are generated for each county.
